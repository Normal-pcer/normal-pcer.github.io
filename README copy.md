## F&Q
- Q:无法访问github，如何下载Wordcraft的其他版本？
```
A: 1.在Gitee仓库的Release中找到你要下载的版本所对应的zip文件(e.g: 0.2.1.zip)
2. 在 安装目录\default\ 下新建文件夹，命名为对应的版本(不带字母v, 如0.2.1)
3. 将zip文件解压到刚刚新建的文件夹中
4. 用文本编辑器打开 安装目录\default\verson.json ，在结尾的"]"前添加下载的版本号(不带字母v)
```
